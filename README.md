**What is this**
GlobalPhrasePack bundles short, practical phrase collections and tiny helper tools focused on clarity and usefulness. Each pack pairs short English phrases with a local language translation, readable transliteration and a one-line usage note. The project targets learners, volunteers, community helpers and anyone who needs fast, shareable text snippets in multiple languages.

- Tiny, optional client utilities: search, copy-to-clipboard, simple formatters (currency, phone, date).
- Community-driven: easy to add new locale packs or improve existing entries.

**Contributing**
- Add a new pack: follow the CSV/JSON schema and include at least 20 useful phrases for the locale.
- Keep translations short and neutral; when possible, add transliteration and one usage note.
- Open a pull request with the new file and a short README update describing the locale.
- For large or sensitive topics (medical/legal), prefer conservative phrasing and request review from native speakers.

**Design & UX notes**

- Highlight usage notes in a subtle grey to avoid clutter.
- Sort entries by everyday relevance: greetings, directions, payments, emergency basics.
